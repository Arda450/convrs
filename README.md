# CLI Data Converter

Ein Command-Line-Interface Tool zum Konvertieren zwischen verschiedenen Datenformaten (JSON, YAML, TOML).

## Voraussetzungen

- Rust (mindestens Version 1.70.0)
- Cargo

## Installation

```bash
cargo build --release
```

## Verwendung

```bash
cargo run
```
